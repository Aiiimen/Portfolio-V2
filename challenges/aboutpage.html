About Page Concept

About page concept with multiple hover transitions & animations.

Source Code

Typefaces: Manrope

Colours: #0033FF #000000

Icons: Iconfinder

The HTML Structure

The whole content is wrapped in a p tag – for the elements that contain hover stats they’re wrapped in a span tag. The span tag is used to group inline-elements. It is used to add a hook to a part of a text or a part of a document.

Styling

all span tags are set to have their position relative therefore all the children can be set to absolute to freely move them around if need be. The absolute element is positioned relative to its first positioned (not static) ancestor element.

Hover & Animations

Below I’ll be explaining how I achieved 3 of the animations and transitions that were used. There’re many other ways to achieve the same thing – these are one of those ways.

CSS Gradient Animation

When the mouse is hovered on Designer text an animation called Gradient is triggered.

This animation is a combination of 5 different keyframes where it transitions infinitely between 4 different colours with ease-in-out over 5 seconds.

Notice the colour at 0% and 100% are the same – this is to create a smooth transition when the animation is looped. If the colours were different, you’ll notice a sudden abrupt in colour transitioning

CSS 360deg Rotate Animation

When the @ sign is on a hover state, an email address rotates around it. This is achieved like the previous animation using keyframes but the transform attribute was used. Note the transition used here is linear where it seems like one continues loop. For example if the timing function was ease you would notice when the animation starts and ends.

CSS Floating Animation

The floating animation was achieved by having 3 images that were modified in Width & Height and position in CSS. This was achieved by placing all the img tags inside a container that positioned relative. By setting the img to an absolute position, it gives you the flexibility to move them around easily.

All the images have the same animation called hearts where the opacity gradually gets set to 1 and back to 0 when it reaches 100%. The movement was achieved by incrementing the bottom value. To make the animation more realistic different delays were applied to individual img tags.
